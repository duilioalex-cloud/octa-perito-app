# Modelo - Laudo Pericial de Engenharia Civil

Este modelo foi preparado para pericias de engenharia civil envolvendo imoveis, obras, vicios construtivos, danos, manifestacoes patologicas, conformidade tecnica, custos e respostas a quesitos.

Ele deve ser adaptado ao caso concreto, aos quesitos admitidos, aos documentos juntados, as normas tecnicas pertinentes e as limitacoes observadas em vistoria.
