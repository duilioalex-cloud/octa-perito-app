# OCTA Perito v0.7.0

## Painel Financeiro e Operacional

Esta versão entrega o painel executivo com filtros combinados por período e processo, indicadores financeiros, diligências, laudos, prazos e pendências.

### Principais alterações

- novo filtro por processo judicial;
- aplicação do escopo selecionado em cards, gráficos, listas e ranking;
- indicador específico de diligências e vistorias;
- separação entre prazos próximos e prazos vencidos;
- prioridades reordenadas por criticidade operacional e financeira;
- manutenção dos indicadores de honorários, despesas, deslocamentos e resultados;
- interface responsiva ajustada para o novo conjunto de filtros.

Não é necessária nova migração SQL.
